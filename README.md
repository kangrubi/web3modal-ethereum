# web3modal ethereum

### 설치

```
npm install @web3modal/ethereum @web3modal/react wagmi ethers@^5
```

### 프로젝트

https://cloud.walletconnect.com/app
로그인하여 프로젝트를 생성(또는 기존 사용)하고 연결된 프로젝트 ID를 복사한다

### walletConnect 문서

https://docs.walletconnect.com/2.0/
